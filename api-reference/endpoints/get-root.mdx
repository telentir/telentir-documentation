### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/root` |

Returns the per-user root object via `storeService.rootObject`.

## Authorization

- Requires bearer token or API key.
- Phone verification enforced via `objectAuth` middleware.

## Body

_None._

## Response

```jsonc
{
  "stores": {
    "contacts": { "id": "obj_contacts", "defaultKey": "key_contacts" },
    "calls": { "id": "obj_calls", "defaultKey": "key_calls" }
  },
  "servers": {
    "primary": { "publicKey": "-----BEGIN PUBLIC KEY-----..." }
  }
}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/root \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
