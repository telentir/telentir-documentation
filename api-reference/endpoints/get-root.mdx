title: 'GET /api/root'
description: 'Fetch the encrypted root object'

---

## Try it

```bash
curl https://app.telentir.com/api/root \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Root object describing stores and servers
