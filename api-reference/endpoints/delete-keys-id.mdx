### Endpoint

| Method | Path |
| ------ | ---- |
| `DELETE` | `/api/keys/{id}` |

Removes the key and prevents future decrypt/encrypt operations with it.

## Authorization

- Requires bearer token / API key.
- Phone verification enforced.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Key identifier. |

## Body

_Not applicable._

## Response

HTTP `204 No Content`.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/keys/key_123 \
  -X DELETE \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
