### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/objects/{id}` |

Returns the stored ciphertext + auth tag as hex strings.

## Authorization

- Requires bearer token / API key.
- Phone verification enforced.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Object ID returned from `/api/objects`. |

## Body

_Not applicable._

## Response

```jsonc
{
  "id": "obj_123",
  "key_id": "key_123",
  "related_object_id": "obj_parent",
  "content": "4f2a...",
  "auth_tag": "9caa...",
  "metadata": {},
  "created_at": "2025-11-13T16:45:00.000Z",
  "updated_at": "2025-11-13T16:45:00.000Z"
}
```

curl https://app.telentir.com/api/objects/obj_123 \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
