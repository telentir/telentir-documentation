## Try it

```bash
curl https://app.telentir.com/api/objects/{id} \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Encrypted object
- **404** - No description
