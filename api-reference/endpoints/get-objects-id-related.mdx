### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/objects/{id}/related` |

Returns the objects linked via `relatedId`.

## Authorization

- Requires bearer token / API key with phone verification.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Parent object ID. |

## Body

_Not applicable._

## Response

```jsonc
[
  {
    "id": "obj_child",
    "key_id": "key_123",
    "content": "4f2a...",
    "auth_tag": "9caa..."
  }
]
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/objects/obj_parent/related \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
