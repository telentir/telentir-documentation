---
title: 'GET /api/objects/{id}/related'
description: 'List related encrypted objects'
openapi: "api-reference/openapi.json GET /api/objects/{id}/related"
---

## Try it

```bash
curl https://app.telentir.com/api/objects/{id}/related \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Array of encrypted objects
