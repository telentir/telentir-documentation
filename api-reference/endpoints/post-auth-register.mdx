### Endpoint

| Method | Path |
| ------ | ---- |
| `POST` | `/api/auth/register` |

Creates a user record via `app/api/auth/register/route.ts`, sends SMS verification, and issues a temporary token.

## Authorization

- No authentication header required.
- Response sets the `token` cookie so the user can finish phone verification.

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `firstName` | string | Yes | Given name used in greetings and audit logs. |
| `lastName` | string | Yes | Family name. |
| `email` | string | Yes | Unique workspace email. |
| `phone` | string | Yes | Mobile number that will receive verification codes. |
| `password` | string | Yes | Chosen password. |
| `confirmPassword` | string | Yes | Must match `password`. |
| `remember` | boolean | Yes | When `true`, keep the user signed in for 30 days. |

## Response

```jsonc
{
  "status": "phone_not_verified",
  "token": "tel_session_jwt"
}
```

A verification SMS is automatically dispatched.

curl https://app.telentir.com/api/auth/register \

    <h4>JavaScript (fetch)</h4>

```javascript
const payload = {
  firstName: 'Ava',
  lastName: 'Nguyen',
  email: 'ops@telentir.com',
  phone: '+15551234567',
  password: 'P@ssw0rd!',
  confirmPassword: 'P@ssw0rd!',
  remember: true,
};
const res = await fetch('https://app.telentir.com/api/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(payload),
  credentials: 'include',
});
const data = await res.json();
```
