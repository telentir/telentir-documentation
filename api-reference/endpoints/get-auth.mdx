description: 'Get auth status'

---

## Try it

```bash
curl https://app.telentir.com/api/auth \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Current auth state
