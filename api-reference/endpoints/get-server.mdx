title: 'GET /api/server'
description: 'Return the current Telentir server identity and public keys.'
---

<div class="endpoint-grid">
  <section>

### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/server` |

Surfaces metadata from `serverManager.self` for client-side encryption helpers.

## Authorization

- Requires bearer token or API key.
- Phone verification enforced.

## Body

_None._

## Response

```jsonc
{
  "current": "primary",
  "keys": {
    "root": {
      "application/pkix-spki": "MIIBIjANBgk...",
      "application/jwk+json": {
        "kty": "RSA",
        "n": "...",
        "e": "AQAB"
      }
    }
  }
}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/server \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
</details>
