description: 'Create an encrypted object'

---

## Try it

```bash
curl https://app.telentir.com/api/objects \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "$ref": "#/components/schemas/EncryptedObjectInput"
}
```

## Response

- **200** - Persisted object
- **400** - No description
