### Endpoint

| Method | Path |
| ------ | ---- |
| `PUT`  | `/api/objects` |

Writes ciphertext to the encrypted repository via `storeService.store.createObject`.

## Authorization

- Requires bearer token / API key with phone verification.
- `objectAuth` also loads the store service and ensures the user owns the key.

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `keyId` | string | Yes | Store key ID (`/api/keys`). |
| `content` | string | Yes | Base64-encoded ciphertext. |
| `authTag` | string | Yes | Base64-encoded auth tag (GCM). |
| `relatedId` | string | No | Optional parent object ID / relation anchor. |

## Response

```jsonc
{
  "id": "obj_123",
  "key_id": "key_123",
  "related_object_id": "obj_parent",
  "metadata": {},
  "created_at": "2025-11-13T16:45:00.000Z",
  "updated_at": "2025-11-13T16:45:00.000Z"
}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/objects \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "keyId": "key_123",
    "content": "b64cipher==",
    "authTag": "b64tag==",
    "relatedId": "obj_parent"
  }'
```

```javascript
const payload = {
  keyId: 'key_123',
  content: Buffer.from(cipher).toString('base64'),
  authTag: Buffer.from(tag).toString('base64'),
};
const res = await fetch('https://app.telentir.com/api/objects', {
  method: 'PUT',
  headers: {
    Authorization: `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify(payload),
});
const created = await res.json();
```
