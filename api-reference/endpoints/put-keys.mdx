description: 'Create or upload a store key'
openapi: "api-reference/openapi.json PUT /api/keys"
---

## Try it

```bash
curl https://app.telentir.com/api/keys \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "type": "object",
  "required": [
    "server"
  ],
  "properties": {
    "server": {
      "type": "string"
    },
    "key": {
      "type": "string",
      "description": "Base64 encoded AES key (optional)"
    },
    "iv": {
      "type": "string",
      "description": "Base64 encoded IV (required if key is provided)"
    }
  }
}
```

## Response

- **200** - Key ID or published key details
- **400** - No description
