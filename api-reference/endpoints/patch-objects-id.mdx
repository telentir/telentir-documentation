  <section>

### Endpoint

| Method | Path |
| ------ | ---- |
| `PATCH` | `/api/objects/{id}` |

Updates the content for an object without changing its ID.

## Authorization

- Requires bearer token / API key with phone verification.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Object ID to overwrite. |

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `keyId` | string | Yes | Store key to encrypt with. |
| `content` | string | Yes | Base64 ciphertext. |
| `authTag` | string | Yes | Base64 auth tag. |

## Response

HTTP `204 No Content`.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/objects/obj_123 \
  -X PATCH \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"keyId":"key_123","content":"newCipher==","authTag":"newTag=="}'
```
