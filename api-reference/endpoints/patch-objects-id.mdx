## Try it

```bash
curl https://app.telentir.com/api/objects/{id} \
  -X PATCH \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "$ref": "#/components/schemas/EncryptedObjectInput"
}
```

## Response

- **204** - Updated
- **404** - No description
