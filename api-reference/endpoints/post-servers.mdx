## Try it

```bash
curl https://app.telentir.com/api/servers \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "type": "object",
  "required": [
    "name",
    "mode"
  ],
  "properties": {
    "name": {
      "type": "string"
    },
    "mode": {
      "type": "string",
      "enum": [
        "generate",
        "upload_public"
      ]
    },
    "publicKey": {
      "type": "string",
      "description": "Base64URL encoded SPKI string (required when uploading your own key)"
    }
  }
}
```

## Response

- **200** - Server record and public key material
- **400** - No description
