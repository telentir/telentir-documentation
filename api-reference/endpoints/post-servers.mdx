### Endpoint

| Method | Path |
| ------ | ---- |
| `POST` | `/api/servers` |

Create a server entry either by generating a key pair or uploading a public key.

## Authorization

- Requires bearer token.
- Phone verification enforced.

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `name` | string | Yes | Friendly label for the server. |
| `mode` | string | Yes | Either `generate` or `upload_public`. |
| `publicKey` | string | Required when `mode === "upload_public"`; base64url SPKI. |

## Response

```jsonc
{
  "name": "primary",
  "publicKey": "MIIBIjANBgk...",
  "privateKey": "-----BEGIN PRIVATE KEY-----..."
}
```

`privateKey` is returned only when `mode` is `generate`.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/servers \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"primary","mode":"generate"}'
```
