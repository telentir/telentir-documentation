### Endpoint

| Method | Path |
| ------ | ---- |
| `DELETE` | `/api/objects/publish/{type}/{relatedId}` |

Removes the published mapping but leaves the encrypted object untouched.

## Authorization

- Requires bearer token / API key with phone verification.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `type` | string | Collection name. |
| `relatedId` | string | Unique publish key. |

## Body

_Not applicable._

## Response

HTTP `204 No Content`.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/objects/publish/contacts/contact_123 \
  -X DELETE \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
