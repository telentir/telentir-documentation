title: 'DELETE /api/objects/publish/{type}/{relatedId}'
description: 'Unpublish an object'
openapi: "api-reference/openapi.json DELETE /api/objects/publish/{type}/{relatedId}"
---

## Try it

```bash
curl https://app.telentir.com/api/objects/publish/{type}/{relatedId} \
  -X DELETE \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **204** - Unpublished
