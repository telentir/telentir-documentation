### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/servers` |

Fetches the list of servers registered through `createRootServerKeyPair` / `createRootServerWithPublicKey`.

## Authorization

- Requires bearer token.
- Phone verification enforced.

## Body

_None._

## Response

```jsonc
[
  {
    "name": "primary",
    "publicKey": "MIIBIjANBgk..."
  }
]
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/servers \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
</details>
