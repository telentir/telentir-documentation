---
title: 'GET /api/servers'
description: 'List registered root servers'
openapi: "api-reference/openapi.json GET /api/servers"
---

## Try it

```bash
curl https://app.telentir.com/api/servers \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Registered BYOK servers
