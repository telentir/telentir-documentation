description: 'Sign in with email & password'

---

## Try it

```bash
curl https://app.telentir.com/api/auth/login \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "type": "object",
  "required": [
    "email",
    "password",
    "remember"
  ],
  "properties": {
    "email": {
      "type": "string",
      "format": "email"
    },
    "password": {
      "type": "string",
      "format": "password"
    },
    "remember": {
      "type": "boolean",
      "default": false
    }
  }
}
```

## Response

- **200** - Login status
- **400** - No description
