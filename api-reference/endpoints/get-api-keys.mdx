---
title: 'GET /api/api-keys'
description: 'List API keys'
openapi: "api-reference/openapi.json GET /api/api-keys"
---

## Try it

```bash
curl https://app.telentir.com/api/api-keys \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Keys owned by the current user
