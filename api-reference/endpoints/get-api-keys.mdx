<div class="endpoint-grid">
  <section>

### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/api-keys` |

Reads API key metadata from `database.apiKeys.list`.

## Authorization

- Requires bearer token or API key.
- Phone verification not required, but only verified users can create/delete keys later.

## Body

_None._

## Response

```jsonc
[
  {
    "id": "key_123",
    "name": "backend-job",
    "last_used_at": "2025-11-10T18:11:07.000Z",
    "created_at": "2025-11-01T15:00:00.000Z"
  }
]
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/api-keys \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
</details>
