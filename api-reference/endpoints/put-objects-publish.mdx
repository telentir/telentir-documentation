## Try it

```bash
curl https://app.telentir.com/api/objects/publish/{type}/{relatedId} \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "$ref": "#/components/schemas/PublishRequest"
}
```

## Response

- **204** - Published
- **400** - No description
