### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/auth/logout` |

Flushes the `token` cookie and returns a simple status payload.

## Authorization

- Requires a valid session cookie or bearer token so the logout can target the right user.
- Phone verification not required.

## Body

_Not applicable._

## Response

```json
{"status":"logout"}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/auth/logout \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
</details>

  </section>
  <aside>
    <h3>Examples</h3>
    <h4>cURL</h4>

```bash
curl https://app.telentir.com/api/auth/logout \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

    <h4>JavaScript (fetch)</h4>

```javascript
await fetch('https://app.telentir.com/api/auth/logout', {
  headers: { Authorization: `Bearer ${token}` },
  credentials: 'include',
});
```

    <h4>Sample response</h4>

```json
{"status":"logout"}
```
