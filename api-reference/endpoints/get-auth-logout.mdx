<Endpoint method="GET" path="/api/auth/logout" summary="Get Auth Logout" />

<div class="endpoint-grid">
  <section>

### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/auth/logout` |

Flushes the `token` cookie and returns a simple status payload.

## Authorization

- Requires a valid session cookie or bearer token so the logout can target the right user.
- Phone verification not required.

## Body

_Not applicable._

## Response

```json
{"status":"logout"}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/auth/logout \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
</details>

  </section>
