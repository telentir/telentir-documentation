## Try it

```bash
curl https://app.telentir.com/api/api-keys \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "$ref": "#/components/schemas/ApiKeyCreateRequest"
}
```

## Response

- **200** - New key (plaintext is returned once)
- **400** - No description
