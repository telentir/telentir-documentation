### Endpoint

| Method | Path |
| ------ | ---- |
| `PUT`  | `/api/api-keys` |

Generates a key with the `tel_` prefix via `database.apiKeys.create`.

## Authorization

- Requires bearer token or API key belonging to a phone-verified user (`auth({ requirePhoneVerification: true })`).

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `name` | string | Yes | Friendly name to display in the dashboard. |

## Response

```jsonc
{
  "id": "key_123",
  "name": "backend-job",
  "key": "tel_a1b2c3...",
  "last_used_at": null,
  "created_at": "2025-11-13T16:30:00.000Z"
}
```

The `key` value is only returned once.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/api-keys \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"backend-job"}'
```
</details>

    <h3>Examples</h3>
    <h4>cURL</h4>

```bash
curl https://app.telentir.com/api/api-keys \
  -X PUT \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"backend-job"}'
```

    <h4>JavaScript (fetch)</h4>

```javascript
const res = await fetch('https://app.telentir.com/api/api-keys', {
  method: 'PUT',
  headers: {
    Authorization: `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ name: 'backend-job' }),
});
const data = await res.json();
```
