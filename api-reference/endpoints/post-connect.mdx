### Endpoint

| Method | Path |
| ------ | ---- |
| `POST` | `/api/connect` |

Kicks off a streaming job defined in `app/api/connect/route.ts`.

## Authorization

- Requires bearer token / API key with phone verification (object access + billing enforcement).

## Body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `repository` | string | Yes | One of `contacts`, `calls`, `agents`, `tasks`, `conversational-pathways`, `campaigns`, `sms-messages`, `performance-metrics`. |
| `path` | string | Yes | Connector path (e.g., `/scrape-crm`). `/get-leads` triggers lead credit checks. |
| `params` | object | Yes | Parameters forwarded to the connector service. |

## Response

HTTP `204 No Content` indicates the job started (object creation happens asynchronously). When no lead allowance remains, the API also returns `204` after logging.

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/connect \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "repository": "contacts",
    "path": "/sync",
    "params": { "source": "airtable" }
  }'
```
</details>

    <h3>Examples</h3>
    <h4>cURL</h4>

```bash
curl https://app.telentir.com/api/connect \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"repository":"contacts","path":"/get-leads","params":{"credits":25}}'
```

    <h4>JavaScript (fetch)</h4>

```javascript
await fetch('https://app.telentir.com/api/connect', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ repository: 'contacts', path: '/get-leads', params: { credits: 25 } }),
});
```

    <h4>Sample response</h4>

```text
204 No Content
```
