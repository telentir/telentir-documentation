## Try it

```bash
curl https://app.telentir.com/api/connect \
  -X POST \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
  -H "Content-Type: application/json" \
  -d '{ }'
```

## Body

**Required:** yes
- `application/json` body schema:
```json
{
  "$ref": "#/components/schemas/ConnectRequest"
}
```

## Response

- **204** - Job accepted
