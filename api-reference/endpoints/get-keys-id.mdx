## Try it

```bash
curl https://app.telentir.com/api/keys/{id} \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```

## Body

_This endpoint does not accept a request body._

## Response

- **200** - Stored key information
- **404** - No description
