### Endpoint

| Method | Path |
| ------ | ---- |
| `GET`  | `/api/keys/{id}` |

Returns the AES key + IV for a specific store key.

## Authorization

- Requires bearer token / API key.
- Phone verification enforced.

## Path parameters

| Param | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Key identifier returned by `/api/keys`. |

## Body

_Not applicable._

## Response

```jsonc
{
  "id": "key_123",
  "server": "primary",
  "key": "a1b2c3...",
  "iv": "d4e5f6...",
  "deprecated": false
}
```

<details class="try-it">
  <summary>Try it</summary>

```bash
curl https://app.telentir.com/api/keys/key_123 \
  -H "Authorization: Bearer $TELENTIR_TOKEN"
```
